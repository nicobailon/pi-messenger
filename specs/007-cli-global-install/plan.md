# Plan: CLI Global Install via jiti Wrapper

## Summary

Make `pi-messenger-cli` globally available by writing a shell wrapper to `~/.pi/agent/bin/` during `install.mjs`. The wrapper invokes pi's built-in jiti to run the CLI's TypeScript source directly. Zero new dependencies, zero bundling, changes are immediately live.

### 3. install.mjs — Jiti resolver + wrapper function (before collision guard)

```js
function resolveJitiPath() {
  try {
    const npmPrefix = execFileSync("npm", ["prefix", "-g"], { encoding: "utf-8" }).trim();
    const jitiPath = path.join(npmPrefix, "lib", "node_modules",
      "@mariozechner", "pi-coding-agent", "node_modules",
      "@mariozechner", "jiti", "lib", "jiti-cli.mjs");
    if (fs.existsSync(jitiPath)) return jitiPath;
  } catch {}
  return null;
}

function installCliWrapper(sourceDir) {
  const jitiPath = resolveJitiPath();
  if (!jitiPath) {
    console.error("⚠ CLI wrapper not created: pi not found (jiti unavailable).");
    console.error("  Install pi first, then re-run: npx pi-messenger");
    process.exitCode = 1;
    return false;
  }
  fs.mkdirSync(BIN_DIR, { recursive: true });
  const wrapper = `#!/bin/bash
# Generated by pi-messenger install — do not edit
JITI_PATH="${jitiPath}"
SOURCE_DIR="${sourceDir}"
if [ ! -f "$JITI_PATH" ]; then
  echo "pi-messenger-cli: jiti not found at $JITI_PATH" >&2
  echo "Pi may have been reinstalled. Re-run: npx pi-messenger" >&2
  exit 1
fi
if [ ! -f "$SOURCE_DIR/cli/index.ts" ]; then
  echo "pi-messenger-cli: source not found at $SOURCE_DIR/cli/index.ts" >&2
  echo "Re-run: npx pi-messenger" >&2
  exit 1
fi
exec node "$JITI_PATH" "$SOURCE_DIR/cli/index.ts" "$@"
`;
  fs.writeFileSync(CLI_WRAPPER_PATH, wrapper, { mode: 0o755 });
  return true;
}
```

### 4. install.mjs flow — wrapper before collision guard, guard exits 0

```
1. Parse args, load package.json
2. Handle --help, --crew-install, --crew-uninstall
3. Handle --remove (unconditional wrapper + extension cleanup)
4. Handle "already in extension dir"
5. installCliWrapper(PACKAGE_DIR) — runs unconditionally
6. Collision guard:
   - If collision AND wrapper succeeded → exit 0 with note
   - If collision AND wrapper failed → exit 1 with error
7. Extension copy (copyDir) — re-run installCliWrapper(EXTENSION_DIR)
8. Post-install output (includes CLI path)
```
